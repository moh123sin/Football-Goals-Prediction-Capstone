This project predicts the number of goals scored in Premier League football matches using historical match data and machine learning models. I cleaned and prepared the raw datasets, engineered match‑level and team‑level features, and trained predictive models to estimate expected goals for each fixture, then evaluated their performance on hold‑out data.
The repository includes separate folders for data, models and notebooks, along with a Power BI dashboard that visualizes predictions, team performance and goal trends across seasons. The final report and video explanation summarize the modelling approach, highlight key factors that drive goal outcomes and show how this solution can support match analysis, betting strategies or fan insights.
